Weather History Visualizer

A Python-based desktop application that loads historical weather data and generates multiple visualizations, including heatmaps, bar charts, scatter plots and windspeed analysis.
Built using Tkinter, Pandas, NumPy, Matplotlib and Seaborn.

Features:-

1. Load any CSV weather dataset

2. Automatic date parsing (extracts year, month, day)

3. Data cleaning and preprocessing

4. Scrollable table view of dataset

5. Monthly average temperature heatmap

6. Annual windspeed dot plot

7. Precipitation scatter plot

8. Yearly average temperature bar chart

9. Fully embedded visualizations inside Tkinter GUI

How to Run:

1. Install required libraries:

     "pip install pandas numpy matplotlib seaborn"
   
2. Run the main file:
   
     "python weather_app_improved.py"
